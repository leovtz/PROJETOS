# Manipulação de Dados

## Descrição
Este projeto demonstra como manipular dados usando Python e SQL. Um banco de dados SQLite é criado e alimentado com dados de vendas. O projeto inclui a realização de consultas SQL para calcular métricas importantes como total de vendas e média, além de exportar os resultados para um arquivo CSV.

## Tecnologias Utilizadas
- Python
- SQLite
- Pandas
